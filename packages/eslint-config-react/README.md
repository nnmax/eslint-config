# @nnmax/eslint-config-react

> See [@nnmax/eslint-config/README.md](https://github.com/nnmax/eslint-config/blob/main/README.md) for more information.

## Install

```bash
npm i -D @nnmax/eslint-config-react
```

## Configuration

Adds `"extends": ["@nnmax/eslint-config-react"]` to your `.eslintrc`.
